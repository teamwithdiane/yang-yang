# Garden Oasis of San Jacinto Assisted Living Facility

A light, luxurious website for Garden Oasis of San Jacinto Assisted Living Facility.

## Project Structure

- `index.html` - Main homepage
- `amenities.html` - Amenities page
- `css/` - Stylesheets
- `js/` - JavaScript files
- `images/` - Image assets

## Features

- Responsive design
- Light, luxurious theme
- Amenities showcase section
- Contact information
- Accessibility compliant

## Getting Started

Open `index.html` in your browser to view the website.
