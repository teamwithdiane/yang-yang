# Garden Oasis of San Jacinto — Website

A single-page website for Garden Oasis of San Jacinto Assisted Living Facility.

## What's in here
- `index.html` — the entire site (HTML + CSS in one file, no build tools needed)

## How to put this on GitHub and make it live (free)

1. **Create a repository**
   - Go to github.com → click the `+` in the top right → **New repository**
   - Name it something like `garden-oasis-website` → Create repository

2. **Upload the file**
   - On the new repo page, click **Add file → Upload files**
   - Drag in `index.html` (and this `README.md`) → **Commit changes**

3. **Turn on GitHub Pages**
   - Go to **Settings → Pages** (left sidebar)
   - Under "Build and deployment," set Source to **Deploy from a branch**
   - Branch: `main`, folder: `/(root)` → **Save**

4. **Visit your site**
   - GitHub will show a URL like `https://yourusername.github.io/garden-oasis-website/`
   - It usually goes live in 1–2 minutes

## Editing later
Everything — text, phone number, address, colors — lives in `index.html`. Open it in any text editor (even GitHub's own editor, the pencil icon on the file page), change the words between the tags, and commit again. The site updates automatically.

## Before you launch for real
- Swap the placeholder phone number, address, and email in `index.html` for your real ones
- Replace the license line in the footer with your actual facility license number
- Consider a real domain name (e.g. gardenoasissj.com) pointed at GitHub Pages, or hosting through a provider that also handles email
