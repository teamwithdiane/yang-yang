# Images Directory

To enhance the website with real caregiving images, please add the following image files to this directory:

## Recommended Images:

1. **caregiver-helping.jpg** - Professional caregiver assisting a senior resident
2. **health-checkup.jpg** - Healthcare professional checking resident's vitals
3. **social-activity.jpg** - Residents enjoying a group activity or event
4. **personalized-care.jpg** - One-on-one care moment

## How to Add Images:

1. Save your images in this directory (`images/`)
2. Update the HTML file to reference these images
3. Images should be high quality and at least 800x600 pixels
4. Use JPG or PNG format
